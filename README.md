## 📚 Projetos principais

### [clinic-management-api](https://github.com/luisgabrielMR/clinic-management-api)

API REST em desenvolvimento para gestão simplificada de uma clínica de fisioterapia, criada como projeto público de portfólio.

O objetivo é demonstrar fundamentos de backend com ASP.NET Core, PostgreSQL, documentação Swagger, Docker Compose, CRUDs, validações básicas e regras de negócio. A proposta é ser uma versão menor, honesta e bem acabada de um sistema de clínica, sem complexidade desnecessária.

**Tecnologias e conceitos:** C#, ASP.NET Core, PostgreSQL, Entity Framework Core, API REST, Docker, Swagger, regras de negócio.

### [Graph-relation-finder](https://github.com/luisgabrielMR/Graph-relation-finder)

Projeto em C# baseado na teoria dos seis graus de separação. Utiliza grafos para representar relações entre atores e filmes, permitindo buscar conexões a partir de dados carregados de um arquivo JSON.

O projeto demonstra estrutura de dados, busca em largura, busca em profundidade limitada e organização de lógica para consulta de relações.

**Tecnologias e conceitos:** C#, .NET, grafos, BFS, DFS, JSON, API REST, estrutura de dados.

### [rpg-battle-grpc](https://github.com/luisgabrielMR/Sistema-de-batalha-RPG-distribu-do)

Sistema distribuído acadêmico com comunicação gRPC entre um cliente Python e um servidor Node.js. O projeto simula uma batalha de RPG e também utiliza transferência de arquivos por streaming.

Esse projeto demonstra comunicação backend-to-backend, uso de contrato `.proto`, separação entre cliente e servidor e integração entre tecnologias diferentes.

**Tecnologias e conceitos:** gRPC, Protocol Buffers, Node.js, Python, sistemas distribuídos, streaming de arquivos, backend-to-backend.

## 🧠 Projetos acadêmicos complementares

### [Paralelismo](https://github.com/luisgabrielMR/Paralelismo)

Projeto acadêmico para comparar estratégias sequenciais e paralelas de busca de nomes em arquivos `.txt`.

A aplicação usa Java para executar as buscas e medir o tempo de execução, enquanto scripts em Python automatizam benchmarks, geram arquivos CSV/JSON e produzem relatórios para análise dos resultados.

**Tecnologias e conceitos:** Java, Python, paralelismo, threads, benchmark, análise de desempenho, relatórios.

### [Compiladores](https://github.com/luisgabrielMR/Compiladores)

Projeto acadêmico de compiladores para uma linguagem simplificada, envolvendo análise léxica, análise sintática, análise semântica, geração de tokens em JSON e relatórios HTML.

**Tecnologias e conceitos:** Python, compiladores, análise léxica, análise sintática, análise semântica, LL(1), relatórios HTML.

### [sistema-especialista](https://github.com/luisgabrielMR/sistema-especialista)

Sistema especialista baseado em regras de produção, utilizando CLIPS integrado ao Python para diagnóstico de incidentes de infraestrutura de TI a partir de alertas e métricas.

**Tecnologias e conceitos:** Python, CLIPS, sistemas especialistas, regras de produção, diagnóstico, explicabilidade.
