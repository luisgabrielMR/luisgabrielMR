## 📚 Projetos em destaque

### [Gestao_Clinica_Fisioterapia](https://github.com/luisgabrielMR/Gestao_Clinica_Fisioterapia)

Projeto voltado ao desenvolvimento de um sistema para gestão de clínica, com foco em backend, banco de dados e organização das regras do sistema. O projeto envolve conceitos importantes para aplicações reais, como cadastro de entidades, estruturação de dados e evolução da arquitetura.

**Tecnologias e conceitos:** C#, backend, banco de dados, modelagem, organização de sistema.

### [Graph-relation-finder](https://github.com/luisgabrielMR/Graph-relation-finder)

Projeto em C# baseado na teoria dos seis graus de separação. Utiliza grafos não direcionados para representar relações entre atores e filmes, permitindo buscas de caminhos a partir de dados carregados de um arquivo JSON.

**Tecnologias e conceitos:** C#, grafos, busca em largura, JSON, estrutura de dados.

### [ApiGestaoEnsino](https://github.com/KarenStackoski/ApiGestaoEnsino)

API para gestão de ensino, desenvolvida com JavaScript. Projeto acadêmico voltado à criação de endpoints e organização de funcionalidades para gerenciamento de dados.

**Tecnologias e conceitos:** JavaScript, API REST, backend, organização de rotas.

### [Wine](https://github.com/luisgabrielMR/Wine)

Aplicativo Android desenvolvido em Java, com foco em cadastro e gerenciamento de informações relacionadas a vinhos e vinícolas. O projeto explora persistência local e organização de telas em uma aplicação mobile.

**Tecnologias e conceitos:** Java, Android, Room, persistência local.

### [Robo-Sumo](https://github.com/luisgabrielMR/Robo-Sumo)

Projeto desenvolvido para a disciplina de Sistemas Embarcados, com programação em C++ aplicada ao controle de um robô sumô.

**Tecnologias e conceitos:** C++, sistemas embarcados, lógica de controle.

### [Aprendendo_Nova_Linguagem_Cs](https://github.com/luisgabrielMR/Aprendendo_Nova_Linguagem_Cs)

Repositório de estudos em C#, usado para praticar fundamentos da linguagem, lógica de programação e conceitos importantes para desenvolvimento backend.

**Tecnologias e conceitos:** C#, fundamentos, prática de programação.
